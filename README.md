# Meme Database

Meme Database is a backend focused project that can be used to store
memes in a database. The memes can be seacrhed using tags, titles or IDs.

**Note:** The project is currenlty under development. Basic logic works
as intended, but needs few adjustments. However, the graphical design is
ugly, so it needs a lot of work.


## Project Overview

The application allows user to:
- Store memes with associated metadata (title, tags, auto generated ID)
- Allowed meme types: images and GIFs
- Search memes by tags
- Search memes by title or ID

The Electron is used for the GUI. 


## Built with 

- **Backend:** Java
- **Database:** SQLite (Java)
- **Frontend:** JavaScript, HTML, CSS 


## Usage

1. Clone the repository:
```bash
git clone https://github.com/turvasa-ko/meme-database.git       
```

2. Run the application:
```bash
python run.py
```
